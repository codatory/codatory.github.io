Following Heroku's [unceremonious announcement](https://blog.heroku.com/next-chapter) 
of the end of free services effective September 28th, I have updated the readme for 
the [IndyHackers Planet repository](https://github.com/codatory/indyhackers.org-planet) 
announcing the end of the service to coincide.

If you were still relying on the aggregated RSS feed, now is a suitable time to grab 
the OPML and do the aggregation yourself. I would not recommend hosting your own copy,
given how old and unmaintained the code is.
